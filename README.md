# Etch-a-Sketch (The Odin Project)

## Requirements implemented
- **16x16 grid** generated in JavaScript (no manual div copying in HTML).
- **Flexbox grid** using a fixed-size container (960px) centered on the page.
- **Dynamic square sizing**: each square is sized in JS based on `CONTAINER_SIZE / gridSize`.
- **Sketching effect**: squares turn on when you hover (`mouseenter`).
